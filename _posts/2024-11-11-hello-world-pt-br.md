# Olá, Mundo! Em 7 Linguagens

Bem-vindos, exploradores do código! 🌍 Nenhuma jornada de programação está completa sem o humilde "Olá, Mundo!" — o equivalente a dizer "Oi, estou aqui!" em uma nova linguagem. Hoje, vamos ver o "Olá, Mundo!" em sete linguagens diferentes. Viva o primeiro print da nossa vida!

---

## C

Ah, C! A espinha dorsal de muitos sistemas. Veja como C diz "Olá, Mundo!":

```c
#include <stdio.h>

int main() {
    printf("Olá, Mundo!\n");
    return 0;
}
```

## C#

O C# facilita tudo com poucas linhas, usando o framework .NET.

```c#
using System;

class Program {
    static void Main() {
        Console.WriteLine("Olá, Mundo!");
    }
}
```

## Ruby

Ruby mantém tudo simples e elegante. Essa linha é o mínimo possível!

```ruby
puts "Olá, Mundo!"
```

## Go

Go é robusto e opinativo, mas conciso. Veja o "Olá, Mundo!" em Go:

```go
package main

import "fmt"

func main() {
    fmt.Println("Olá, Mundo!")
}
```

## Node.js

JavaScript rodando no servidor? Com certeza! Veja como fazer com Node.js:

```javascript
console.log("Olá, Mundo!");
```

## Python

A sintaxe do Python é famosa pela facilidade para iniciantes. Veja como fica:

```python
print("Olá, Mundo!")
```

## Rust

Rust é poderoso e seguro. Ele te mantém alerta, mas te recebe de braços abertos:

```rust
fn main() {
    println!("Olá, Mundo!");
}
```

E aí está! Sete linguagens, sete saudações, uma mensagem universal: "Olá, Mundo!" Independente da linguagem escolhida, cada primeiro programa nos lembra da maravilha e criatividade que existe na programação. Então, qual é o seu "Olá, Mundo!" favorito?

Bom código! 😃
